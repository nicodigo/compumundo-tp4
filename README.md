# README — Trabajo Práctico 4

## Sistemas Distribuidos y Programación Paralela

### Universidad Nacional de Luján

Este repositorio contiene el desarrollo del Trabajo Práctico 4 de la materia Universidad Nacional de Luján, enfocado en procesamiento gráfico, shaders y manipulación de imágenes mediante técnicas de renderizado y programación paralela sobre GPU.

---

# Objetivos

* Comprender el funcionamiento del pipeline gráfico.
* Analizar distintos tipos de shaders y su aplicación.
* Implementar transformaciones y efectos visuales sobre imágenes.
* Explorar procesamiento paralelo aplicado al renderizado.
* Experimentar con operaciones de fragment shaders y manipulación de texturas.
